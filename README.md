## 🚀 Getting Started

### Prerequisites

* Flutter SDK (latest stable)
* Android Studio / VS Code
* Dart plugin installed
* Emulator / Physical device

### Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/rohanp12326/exp_flutter.git
   cd flutter-project
   ```
2. Install dependencies:

   ```bash
   flutter pub get
   ```
3. Run the app:

   ```bash
   flutter run
   ```
